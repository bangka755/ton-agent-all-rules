> Next-generation AI agents platform built on the TON blockchain ecosystem.

## 🚀 Features

- **TON Connect 2.0** — Seamless wallet connection (Tonkeeper, OpenMask, MyTonWallet)
- **TON MCP Integration** — Model Context Protocol for AI-chain interaction
- **Agentic Wallet** — Programmable AI-controlled wallets
- **TON Sites** — Decentralized hosting
- **TON Storage** — Persistent agent memory
- **TON DNS** — Human-readable .ton addresses
- **TON Payment Channels** — Instant micropayments
- **Mini Games** — Memory Match, TON Miner, TON Quiz

## 📦 Deployment

### GitHub Pages
1. Fork this repo
2. Go to Settings → Pages → Source: main branch
3. Update `tonconnect-manifest.json` with your GitHub Pages URL
4. Your site is live!

### Manual
Simply serve `index.html` via any static hosting.
